NGINX Unit Ansible Collection
=============================

NGINX Unit Ansible Collection is a bundle of Ansible content that we can use
to install Unit and manage configuration for already running instances of
Unit.

Collection is freely available on `Ansible Galaxy`_.

.. _Ansible Galaxy:
   https://galaxy.ansible.com/steampunk/unit
